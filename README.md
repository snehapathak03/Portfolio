## PERSONAL PORTFOLIO WEBSITE

A modern, responsive, and interactive personal portfolio website developed to showcase my technical expertise, projects, education, certifications, 
and professional journey. The website features animated UI components, responsive layouts, project showcases, and an EmailJS-powered contact form, 
providing recruiters and visitors with an engaging way to explore my work and connect with me.

## Features

- Modern and Responsive UI
- Animated Hero Section
- About Me
- Technical Skills with Animated Progress Bars
- Professional Skills Visualization
- Project Showcase
- Education Timeline
- Contact Form using EmailJS
- Social Media Integration
- Smooth Scrolling Navigation

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- EmailJS
- Font Awesome
- Git & GitHub
